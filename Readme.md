# Weather-App 
This is weather web application. Optimised for mobile and Big screens. 
> Skills Used : **HTMl5**, **Tailwindcss**, **Javascript**, **Git**, **Developer Tools**, **Visual Studio Code** etc.
## Installation : You can 'fork' it from the code repo link.  
## Requirements : To run this code on your local computer U need to install certain softwares and frameworks.    
1. **Node.js** - Visit this link to install [Node.js](https://nodejs.org/en/download) on your computer according to your OS.  
2. **npm** - After successfully installing Node.js initialize npm(Node Package Manager) using the command :  
`npm init-y`  
this will create a 'package.json' file.  
3. **Tailwindcss** - Tailwind Css is the postcss framework used to style the page. Visit [Tailwindcss](https://tailwindcss.com/docs/installation) to install the TailwindCss framework.  

   
After that run this commands in the bash terminal : 
``` 
npm install -D tailwindcss
npx tailwindcss init 
npx tailwindcss -i ./style.css -o .dist/output.css -- watch
```
